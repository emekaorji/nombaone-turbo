Create your first plan on the Nomba One sandbox with **cURL**. This is a real
call that returns a `201`; only the key (`nbo_sandbox_…`) and the host change when
you go live.

## 1 · Get a sandbox key

Grab a `nbo_sandbox_…` key. See [authentication](/getting-started/authentication).
Just a shell. Export your key: `export NOMBAONE_API_KEY=nbo_sandbox_…`.
