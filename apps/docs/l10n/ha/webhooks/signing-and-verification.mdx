Kada ka taɓa amincewa da webhook da ba ka tantance ba: URL ɗin a bayyane yake, don haka
kowa na iya yin `POST` zuwa gare shi. Kowace delivery ta gaskiya tana ɗauke da header
ɗin `X-Nombaone-Signature` da za ka kwatanta da signing secret na endpoint ɗinka.

## Sa hannun

Header ɗin shi ne `t=<unix>,v1=<hex>`, inda `v1` yake:

```
key = SHA256( whsec )                       // hex — derive once from your secret
v1  = HMAC_SHA256( `${t}.${rawBody}`, key ) // hex
```

`t` shi ne tambarin lokacin delivery, `rawBody` kuma shi ne **ainihin bytes** na body
ɗin buƙatar. Ka tantance ta hanyar sake ƙididdige `v1` sannan ka kwatanta.

<Callout type="warning" title="Ka sa hannu kan raw bytes: kada ka sake yin serialize">
  Ka ƙididdige HMAC kan body ɗin daidai yadda ka karɓe shi. `JSON.parse` sannan
  `JSON.stringify` na iya sake tsara key ɗin su canza bytes, don haka sa hannun ba
  zai yi daidai ba. Ka kama raw body kafin wani framework ya yi masa parse.
</Callout>

## Misalin aiwatarwa

<CodeGroup>
  <Tab label="TypeScript">
    ```ts
    import { createHash, createHmac, timingSafeEqual } from "node:crypto";

    export function verify(rawBody: string, header: string, secret: string) {
      const parts = Object.fromEntries(header.split(",").map((kv) => kv.split("=")));
      const { t, v1 } = parts;

      // Reject stale deliveries (replay protection).
      const age = Math.abs(Date.now() / 1000 - Number(t));
      if (age > 300) throw new Error("timestamp too old");

      // The HMAC key is the sha256 (hex) of your plaintext whsec.
      const key = createHash("sha256").update(secret).digest("hex");
      const expected = createHmac("sha256", key).update(`${t}.${rawBody}`).digest("hex");
      const ok =
        v1.length === expected.length &&
        timingSafeEqual(Buffer.from(v1), Buffer.from(expected));
      if (!ok) throw new Error("invalid signature");
    }
    ```
  </Tab>
  <Tab label="Python">
    ```python
    import hmac, hashlib, time

    def verify(raw_body: bytes, header: str, secret: str) -> None:
        parts = dict(kv.split("=") for kv in header.split(","))
        t, v1 = parts["t"], parts["v1"]

        if abs(time.time() - int(t)) > 300:
            raise ValueError("timestamp too old")

        # The HMAC key is the sha256 (hex) of your plaintext whsec.
        key = hashlib.sha256(secret.encode()).hexdigest()
        expected = hmac.new(
            key.encode(), f"{t}.".encode() + raw_body, hashlib.sha256
        ).hexdigest()
        if not hmac.compare_digest(v1, expected):
            raise ValueError("invalid signature")
    ```
  </Tab>
</CodeGroup>

## Ƙaʼidodi biyu masu muhimmanci

- **Kwatantawa cikin lokaci mai daidaito.** Ka yi amfani da `timingSafeEqual` /
  `compare_digest`, kada ka taɓa amfani da `===`. Kwatantawa ta kai tsaye tana tona
  asirin byte ɗaya bayan ɗaya a ƙarƙashin harin auna lokaci.
- **Ka ƙi tambarin lokaci da ya tsufa.** Ka iyakance `t` zuwa ʼyan mintuna kaɗan domin
  kada a sake kunna delivery da aka kama a kanka daga baya.
