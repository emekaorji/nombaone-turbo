Kò sí customer tí yóò dúró sórí price kan ṣoṣo títí láé. Wọ́n máa ń gbé e sókè,
wọ́n máa ń sọ̀ ọ́ kalẹ̀, wọ́n máa ń fi ìjókòó kún un, tàbí wọ́n máa ń yí padà láti
oṣooṣù sí ọdọọdún — ó sì sábà máa ń ṣẹlẹ̀ **láàárín cycle**, nígbà tí wọ́n ti san
owó apá kan àkókò náà tán. Ìyípadà plan yóò gbé subscription lọ sí price tuntun,
yóò sì yanjú ìyàtọ̀ owó náà; nítorí pé
[proration jẹ́ ìṣòro ledger](/concepts/hard-parts/proration-is-a-ledger-problem),
a ó ṣírò rẹ̀ ní àwọn ẹsẹ̀ kobo tí ó dọ́gba, kì í ṣe àfojúsùn floating-point.

## Yí price padà

Fi subscription náà sí price mìíràn. Ìpè kan ṣoṣo ni yóò ṣe ìyípadà náà àti
proration rẹ̀:

```bash
curl -X POST https://sandbox.api.nombaone.xyz/v1/subscriptions/{id}/change \
  -H "Authorization: Bearer nbo_sandbox_…" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: $(uuidgen)" \
  -d '{ "priceId": "{newPriceId}", "prorationBehavior": "create_prorations" }'
```

Ẹ̀rọ náà yóò dá owó apá price àtijọ́ tí a kò lò padà, yóò sì gba owó apá price
tuntun tí a ti lò, dé kobo gan-an. Nígbà **ìgbésókè**, customer yóò san ìyàtọ̀ náà
lẹ́sẹ̀kẹsẹ̀; nígbà **ìsọ̀kalẹ̀**, credit náà yóò wọ inú invoice tí ó ń bọ̀.

## Wòye rẹ̀ kí o tó pinnu

Fi iye owó gangan tí ìyípadà kan yóò ná customer hàn án kí o tó ṣe é:

```bash
curl https://sandbox.api.nombaone.xyz/v1/subscriptions/{id}/upcoming-invoice \
  -H "Authorization: Bearer nbo_sandbox_…"
```

Invoice tí ó ń bọ̀ yóò fi àwọn ìlà proration hàn, kí ohunkóhun má bàa yà ẹnikẹ́ni
lẹ́nu nígbà tí owó tí ó kàn bá dé.

## Yí ìlù ìsanwó padà

Yíyí padà láti oṣooṣù → ọdọọdún jẹ́ ìyípadà sí price tí ìlù rẹ̀ yàtọ̀ lórí plan kan
náà. Ìlù jẹ́ **ẹyọ ìwọ̀n ní ìlọ́po iye kan**, nítorí náà oṣooṣù → oṣù mẹ́ta mẹ́ta
(`month` × 3) àti oṣooṣù → gbogbo ìṣẹ́jú mẹ́wàá (`minute` × 10) jẹ́ ìyípadà ìlù pẹ̀lú —
ìyípadà iye náà jẹ́ ìyípadà gidi bí ìyípadà ẹyọ ìwọ̀n náà:

```bash
curl -X POST https://sandbox.api.nombaone.xyz/v1/subscriptions/{id}/change \
  -H "Authorization: Bearer nbo_sandbox_…" \
  -H "Content-Type: application/json" \
  -d '{ "priceId": "{yearlyPriceId}", "intervalSwitch": true }'
```

Ẹ̀rọ náà máa ń mọ ìyípadà ìlù fúnra rẹ̀, láti inú price méjèèjì — o kò ní láti sọ fún un.
Dípò kí ó prorate ọdún kan lórí oṣù kan, ó ń dá owó tí ó kù lórí ìlù àtijọ́ padà gẹ́gẹ́
bí credit, ó ń gba owó sáà tuntun ní kíkún, ó sì ń tún ìdákọ̀ró cycle náà ṣe.
`intervalSwitch: true` ń béèrè fún ìtọ́jú kan náà nígbà tí ìlù náà **kò** yí padà.

## Ṣàkóso ìwà proration

- **`create_prorations`**: ṣírò ìyàtọ̀ náà kí o sì lò ó (èyí ni ìpìlẹ̀ fún àwọn
  ìgbésókè).
- **`none`**: yí price padà láìsí proration kankan; iye tuntun yóò bẹ̀rẹ̀ sí í ṣiṣẹ́
  láti cycle tí ó ń bọ̀.

<Callout type="note" title="Ìyípadà iye náà ń prorate pẹ̀lú">
  Fífi ìjókòó kún un tàbí yíyọ ìjókòó kúrò láàárín cycle jẹ́ ọ̀nà kan náà: fi
  `quantity` ránṣẹ́ sí `change`, ìyàtọ̀ tí ó wà lórí ìjókòó kọ̀ọ̀kan yóò sì prorate
  gẹ́lẹ́ bí ìyípadà price.
</Callout>

<Callout type="tip" title="Ṣètò ìyípadà fún ìsọdọ̀tun tí ó ń bọ̀">
  Láti yẹra fún proration láàárín cycle pátápátá, ṣètò kí ìyípadà náà bẹ̀rẹ̀ ní ààlà
  àkókò tí ó ń bọ̀ pẹ̀lú `POST /v1/subscriptions/{id}/schedule`. Customer yóò parí
  àkókò tí ó ti san owó rẹ̀, yóò sì tún subscription rẹ̀ ṣe lórí price tuntun.
</Callout>
