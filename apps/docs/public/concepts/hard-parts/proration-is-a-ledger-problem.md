# Proration is a ledger problem, not a math problem

> Draft. On the roster, not yet written.

Mid-cycle plan changes are not a formula, they are entries that must balance to the kobo. Proration is integer
arithmetic with a largest-remainder split; a downgrade banks a credit, it does not refund to the rail.
