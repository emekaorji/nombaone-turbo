# Elixir method reference

Every function in the `nombaone` Hex package, grouped by the namespace you reach
it through (`Nombaone.Customers.create`, `Nombaone.Plans.Prices.create`, and so
on). The client is always the first argument, and every function has a raising `!`
variant alongside the `{:ok, _}` / `{:error, _}` form. Open any function for its
full request and response shape, with a copy-runnable sample in Elixir and every
other SDK.
